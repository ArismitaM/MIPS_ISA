# MIPS_ISA

A simple MIPS ISA processor simulator implemented in Python and validated using the **MARS assembler**.

This project simulates execution of MIPS instructions and demonstrates correct processor behavior across multiple assembly programs.

---

## 🧠 Overview

This repository contains:

- A Python-based **MIPS processor simulator** (`Processor.py`)
- Several MIPS assembly programs (`*.asm`)
- Corresponding machine code files
- A project report (`ProjectReport.pdf`)

The simulator reads MIPS instructions, simulates pipeline execution, and validates functional correctness against expected outcomes.

---

## 🚀 Features

- Simulates instruction execution for basic MIPS ISA programs
- Supports assembly programs converted to machine code
- Validated for correctness with multiple test cases
- Easy to extend for additional instructions or pipeline features
